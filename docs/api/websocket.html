Server Notification Websocket

When connecting to the server, each client opens a websocket and listens for notifications from the server. The server emits the following notifications

EVENT nc:delta

The nc:delta event is issued every time the machine tool position changes or material removal occurs. The argument is a delta state object

EVENT nc:feed

The nc:feed event is issued every time the machine tool feedrate changes. The argument is the numeric value of the new feed. 'nc:mtc' (MTstate.js) 'nc:probe' 'nc:qifLoad' (tolerances.js)

EVENT nc:speed

The nc:speed event is issued Defined in state.js.

EVENT nc:spindle

The nc:spindle event is issued every time the machine tool spindle speed changes. The argument is the numeric value of the new spindle speed. Defined in state.js. 'nc:state'
