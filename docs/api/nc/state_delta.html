Get Modified Geometry

GET /v3/nc/state/delta

Getting the delta endpoint will return a scene which has the modified geometry from the most recent material removal.

{
  'project': string,
  'workingstep': string,
  'time_in_workingstep': 0,
  'prev': string,
  'geom': geomRef[]
}
Property Unit Description
Project string The name of the project this keystate applies to.
workingstep string The ID of the workingstep this keystate applies to.
time_in_workingstep N/A Always zero. Unused property.
prev string The keystate this is based off of. Currently unimplemented and always an empty string
geom geomRef[] An array of the changed geometry.
