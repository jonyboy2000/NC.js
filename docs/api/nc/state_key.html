Definition: State.js

Get Complete Scene Geometry

GET /v3/nc/state/key

Returns the most up-to-date ‘full scene’ with all geometry currently shown. The Keystate object is as follows

{
  'project': string,
  'workingstep': string,
  'time_in_workingstep': 0,
  'geom': geomRef[]
}
Property Type Description
Project string The name of the project this keystate applies to.
workingstep string The ID of the workingstep that the keystate applies to.
time_in_workingstep N/A Always 0. Unused property.
geom geomRef[] An array of all the geometry in this scene.
